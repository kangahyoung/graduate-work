N
- 고객의 Needs : 재미/새로움
- 불편 : 익숙하고 비슷한 게임들 쉽게 실증남.
- 고객들에게 불편 해소 또는 Needs 충족이 얼마나 중요한가 : 간단하고 단순한 게임도 기존의 게임들과 다른 새로움이 있으면 인기를 얻는다.
A
- 고객 Needs에 대해 해결해 줄 수 있는 방법 :
새로움 / 게임에 조금 더 몰입할 수 있도록 도와준다.
- Needs를 충족시키는 제품/서비스 :   Vr과 심전도 측정기를 사용하여 게임에 더 몰입할 수 있도록 해준다. 
B
- Approach가 고객에게 제공하는 Benefit : 장치를 사용하는 게임 플레이에 있어서 느끼는 새로움 / Vr을 통해 더 몰입할 수 있고 심전도 측정기를 통해 고객의 긴장 상태를 반영하여 현실감을 높인다.
C
- 현재 존재하는 경쟁자 : 네버마인드
- 경쟁자의 장점 : 바이오피드백 증강 호러 어드벤처 게임으로 특이하게 바이오피드백 추적, 동작 탐지, 안면 인식 센서를 이용하여 플레이 할 수 있는데 플레이어가 공포를 느끼게 될시 배경이 바뀌고 노이즈가 끼는 등 분위기가 점점 더 음산하게 된다.
- 경쟁자의 단점 : 무서운 장면이나 요소가 별로 없다. 센서가 게임 값보다 배는 비싸서 보통은 센서 없이 플레이 한다.
- 경쟁자의 장단점과 비교하여 나은 점 : 경쟁자 보다 저렴하고 간단한 장치를 사용
